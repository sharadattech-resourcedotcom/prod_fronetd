If you are using Chrome browser please read

There are two solutions to download and install application. First is to choose another browser when you tap download button (google chrome will prompt you to choose browser, choose another one ,best would be default android browser - called Internet or just Browser, if you choose google chrome new tab will open with blank page!). Second is to press and hold download button a menu will appear, choose save link (application can be downloaded this way, but it cannot be installed through notification, you have to open it via file manager).
